# Shiny-Histogram-Mean-Calculator-App
The application is built using R Shiny, deployed on shinyapps.io, and documented inside the app as required.
The application is built using R Shiny, deployed on shinyapps.io, and documented inside the app as required. This Shiny application allows users to:

Select a dataset (from the built-in R datasets).

Adjust the number of bins using a slider.

View a histogram of the selected data.

Automatically compute and display the mean of the dataset.

The app also includes an embedded Documentation tab explaining how to use the interface, ensuring the app is easy for beginners.
